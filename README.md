# Лабораторная работа №3. Графический редактор
## Задание
Написать программу, реализующую алгоритмы рисования и закраски. Как минимум 3 разные функции вывода линий и 2 разные функции закраски замкнутых контуров (один из них монотонным цветом, а второй с узором). С помощью данных алгоритмов вывести на экран рисунок.
